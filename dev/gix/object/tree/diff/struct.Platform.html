Like Self::for_each_to_obtain_tree(), but with a reusable resource_cache which is used to perform diffs fast.

Reusing it between multiple invocations saves a lot of IOps as it avoids the creation of a temporary resource_cache that triggers reading or checking for multiple gitattribute files. Note that itâ€™s recommended to call gix_diff::blob::Platform::clear_resource_cache() between the calls to avoid runaway memory usage, as the cache isnâ€™t limited.
