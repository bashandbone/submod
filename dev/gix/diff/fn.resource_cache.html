Return a low-level utility to efficiently prepare a blob-level diff operation between two resources, and cache these diffable versions so that matrix-like MxN diffs are efficient.

repo is used to obtain the needed configuration values. mode determines how the diffable files will look like, and also how fast, in average, these conversions are. attr_stack is for accessing .gitattributes for knowing how to apply filters.
